Tags mais úteis em HTML5

Em HTML5 podemos fazer diversas coisas utilizando tags que são comandos pré determinados com funções específicas. Segue abaixo uma lista com os comandos mais úteis (passe o mouse em cima para mais informações).

  • Parágrafos
  • Quebra de linha
  • Adicionar títulos
  • Adicionar imagens
  • Texto em negrito
  • Criar listas ordenadas
  • Adicionar links
